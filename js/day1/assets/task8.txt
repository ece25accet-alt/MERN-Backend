1.console.log("10" + 5);
1.The Output is 105 since we're adding (concatenating) a String and a Number so instead of adding values it converts 5 into String.
-------------------------------------------------------------------

2.console.log("10" - 5);
2.The Output is 5 since "-" only works on numbers so js turns the String "10" into number 10 and do the subtraction.
-------------------------------------------------------------------

3.console.log(true + 1);
3.The Output is 2 since in numeric operation true means 1, so 1 + 1 returns 2 as resut.
-------------------------------------------------------------------

4.console.log(false + 10);
4.The Output is 10 since the bool false is 0 in numeric terms so 0 + 10 is 10.
-------------------------------------------------------------------

5.console.log(Number("100"));
5.The function Number() is used to convert a value into number so the Output is 100 as a number.
-------------------------------------------------------------------

6.console.log(String(500));
6.The function String() is used to convert a value into String so the Output is 500 as a String.
